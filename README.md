# Hangman Game 🎮
A visually interactive Hangman game developed using Python and Tkinter. 
The game features animated hangman stages, a modern dark-themed UI, 
random word generation from a custom dictionary, and an interactive on-screen keyboard.

## Features
- Interactive graphical UI using Tkinter
- Animated hangman progression
- Random word selection from dictionary
- On-screen keyboard
- Disabled buttons after selection
- Win/Lose popup messages
- Modern dark theme interface

## Technologies Used
- Python
- Tkinter
- File Handling
- Random Module
